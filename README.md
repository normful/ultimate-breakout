# Ultimate Breakout

Think you know breakout? Guess again. Play with your friends in real-time and compete for the highest score in our take of this classic arcade game at http://www.ultimatebreakout.com

## Technologies Used

- Node.js
- Socket.IO
- MongoDB
- Phaser
- Express
- Heroku
